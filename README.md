# TypeScript Challenges

A structured collection of **TypeScript challenges** designed to improve
problem-solving skills and build a strong foundation in TypeScript.

---

## 📌 Project Idea

This repository contains a set of coding challenges written in **TypeScript**, aimed at:

* Improving problem-solving skills
* Writing clean and maintainable code
* Mastering core and advanced TypeScript concepts
* Preparing for technical interviews

---

## 🧠 Levels

Challenges are organized into three levels:

* 🟢 **Easy** → Basic concepts and fundamentals
* 🟡 **Medium** → Intermediate logic and problem-solving
* 🔴 **Hard** → Advanced challenges and real-world scenarios


## 🎯 Goal

The main goal of this repository is to build a strong foundation
in TypeScript through consistent practice, not just theory.

---

## 💡 Notes

* Each challenge focuses on clarity and simplicity
* Solutions can always be improved and optimized
* Emphasis on clean, readable code
