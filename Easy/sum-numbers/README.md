# Sum of Numbers

## 🟢 Problem

Write a function in TypeScript that takes an array of numbers and returns the sum of all elements.

---

## ✨ Requirements

* The function should accept an array of numbers.
* The function should return a number.
* If the array is empty, return `0`.

---

## 🧪 Examples

```ts
sumNumbers([1, 2, 3, 4]); // 10
sumNumbers([10, -2, 5]);  // 13
sumNumbers([]);           // 0
```

---

## 🏁 Conclusion

This problem helps practice basic loops and TypeScript typing.
